# English Learning System

A personalized English learning platform with AI-powered content generation and evaluation.

## Features

- **126-Day Structured Curriculum**: Pre-defined learning path from beginner to advanced
- **AI-Powered Content**: Daily lessons generated by Google Gemini
- **Automatic Revision**: Weekly review days (every 7th day)
- **Progress Tracking**: Complete dashboard showing learning journey
- **Smart Context**: Rolling 6-day window prevents token overflow
- **Personalized Learning**: Adapts to your weak areas and learning velocity

## Quick Start

### Prerequisites

- Node.js 18+
- MongoDB (local or Atlas)
- Google Gemini API key

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
cd server && npm install
cd ../client && npm install
```

3. Configure environment:
```bash
# server/.env
MONGODB_URI=your_mongodb_connection_string
GOOGLE_API_KEY=your_gemini_api_key
JWT_SECRET=your_jwt_secret
PORT=3000

# client/.env
VITE_API_URL=http://localhost:3000
```

4. Start the application:
```bash
# Terminal 1 - Server
cd server
npm start

# Terminal 2 - Client
cd client
npm run dev
```

## Architecture

### Backend (`/server`)
- **Express.js** API server
- **MongoDB** for user state and progress
- **Google Gemini** for content generation and evaluation
- **JWT** authentication

### Frontend (`/client`)
- **React + TypeScript**
- **Vite** build tool
- **TailwindCSS** styling

## Key Components

### Curriculum System
- `server/src/trainer/curriculum.js` - 126-day fixed curriculum
- `server/src/trainer/briefBuilder.js` - Generates daily learning briefs
- `server/src/trainer/dayGenerator.js` - Creates daily content

### Context Management
- `server/src/trainer/contextBuilder.js` - Rolling 6-day context window
- `server/src/trainer/contextValidator.js` - Validates context structure
- `server/src/trainer/tokenBudget.js` - Enforces token limits (780 max)

### Evaluation
- `server/src/trainer/evaluationService.js` - Evaluates submissions
- `server/src/trainer/validator.js` - Validates submission format

### Progress Tracking
- `server/src/trainer/progressService.js` - Computes progress metrics
- `GET /api/progress` - Progress dashboard endpoint

## API Endpoints

### Authentication
- `POST /api/auth/signup` - Create account
- `POST /api/auth/login` - Login
- `GET /api/auth/verify` - Verify token

### Learning
- `GET /api/day` - Get today's lesson
- `POST /api/submit` - Submit work for evaluation
- `GET /api/status` - Get current status
- `GET /api/history` - Get learning history
- `GET /api/progress` - Get progress dashboard

### Progress
- `PATCH /api/day/progress` - Update section progress
- `PATCH /api/day/draft` - Save draft

### Reset
- `POST /api/reset` - Full reset
- `POST /api/reset/today` - Reset today only

## Curriculum Structure

- **18 weeks** × 7 days = 126 days
- **6 learning days** + 1 revision day per week
- **Levels**: Beginner (1-42), Intermediate (43-84), Advanced (85-126)
- **Categories**: Grammar, Tense, Vocabulary, Speaking, Writing, Reading, Revision

## Token Budget

The system maintains a strict token budget to prevent API errors:
- **Learner Identity**: ~80 tokens
- **Curriculum Trajectory**: ~200 tokens (last 6 days)
- **Diagnostic Profile**: ~300 tokens
- **Vocabulary Memory**: ~100 tokens
- **Today's Brief**: ~200 tokens
- **Total Maximum**: 780 tokens
